# dature
基于 Node 的新浪博客抓取程序，抓取数据包含标题、正文、时间、分类、标签、图片，并生成 HTML 文件

## 安装

```bash
npm install -g dature 
```

## 使用

```bash
# 抓取牛根生的博客
dature -u 1263917762 -c 'xxxxxx'

# xxxxxx 为 cookie
```
